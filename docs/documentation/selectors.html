Selectors

The selectors config

{
  selects: {
    selectors: { // just css classes
      selected: 'cff-selected',
      opened: 'cff-opened',
      multiple: 'cff-multiple',
      single: 'cff-single',
      disabled: 'cff-disabled',
      uiOption: 'cff-option',
      uiOptDisabled: 'cff-opt-disabled',
      uiItemSelect: 'cff-item',
      uiGroupClass: 'cff-group',
      uiGroupTitle: 'cff-group-title',
      wrapSelect: 'cff-wrap-select',
      containerOptions: 'cff-container-options',
      containerOptsOverlowed: 'cff-overflowed',
      containerOptsOnTop: 'cff-top',
      containerSelected: 'cff-container-display',
      selectedDisplayed: 'cff-display'
    }
  },
  checkboxesRadios: {
    selectors: { // just css classes
      checked: 'cff-checked',
      disabled: 'cff-disabled',
      wrapCheckRadio: 'cff-wrap-check-radio',
      containerCheckRadio: 'cff-container-check-radio',
      isRadio: 'cff-is-radio',
      isCheckbox: 'cff-is-checkbox',
      uiInput: 'cff-ui-input',
      uiLabel: 'cff-ui-label',
      labelLeft: 'cff-text-left',
      labelRight: 'cff-text-right'
    }
  },
  inputsFile: {
    selectors: { // just css classes
      disabled: 'cff-disabled',
      wrapInputFile: 'cff-wrap-input-file',
      uiInputFile: 'cff-input-file',
      uiInputDisplay: 'cff-input-display',
      uiInputLabel: 'cff-input-label',
      uiSelectButton: 'cff-select-button'
    }
  }
};

The select element selectors

Key Type Default Description
selected string cff-selected

Class applied in the ui selected option(s).
opened string cff-opened

Class applied in a opened ui select.
multiple string cff-multiple

Class applied when the ui select is multiple.
single string cff-single

Class applied when the ui select is not multiple.
disabled string cff-disabled

Class applied when the ui selec is disabled.
uiOption string cff-option

Class applied in each ui option.
uiOptDisabled string cff-opt-disabled

Class applied when a ui option is disabled.
uiItemSelect string cff-item

Class applied in all option container children which is visible. This include the ui optgroup titles.
uiGroupClass string cff-group

Class applied in all ui optgroup.
uiGroupTitle string cff-group-title

Class applied in all ui optgroup title.
wrapSelect string cff-wrap-select

Class applied in the root container of ui elements.
containerOptions string cff-container-options

Class applied in all ui container options.
containerOptsOverlowed string cff-overflowed

Class applied in ui container options when selects.autoHeight is true. This class allows that the container get auto vertical scroll.
containerOptsOnTop string cff-top

Class applied in ui container options when selects.autoPositioning is true. This class allows that the container positions on top if it was necessary.
containerSelected string cff-container-display

Class applied in container display of the selected option.
selectedDisplayed string cff-display

Class applied in the ui element that displays the selected option.

The checkbox and radio elements selectors

Key Type Default Description
checked string cff-checked

Class applied in the checked ui element.
disabled string cff-disabled

Class applied in the disabled ui element.
wrapCheckRadio string cff-wrap-check-radio

Class applied in the root container of the all elements.
containerCheckRadio string cff-container-check-radio

Class applied in the container of the ui elements.
isRadio string cff-is-radio

Class applied in the element when it has a type radio.
isCheckbox string cff-is-checkbox

Class applied in the element when it has a type checkbox.
uiInput string cff-ui-input

Class applied in the ui element that represents the corresponding element.
uiLabel string cff-ui-label

Class applied in the ui element label.
labelLeft string cff-text-left

Class applied when the checkboxesRadios.labelPosition is left.
labelRight string cff-text-right

Class applied when the checkboxesRadios.labelPosition is right.

The inputs file selectors

Key Type Default Description
disabled string cff-disabled

Class applied when the element is disabled.
wrapInputFile string cff-wrap-input-file

Class applied in the root container of the ui element.
uiInputFile string cff-input-file

Class applied in the new ui input file element.
uiInputDisplay string cff-input-display

Class applied in the container that displays the selected file(s).
uiInputLabel string cff-input-label

Class applied in the element that displays the selected file(s).
uiSelectButton string cff-select-button

Class applied in the ui button that open the chooser files.
