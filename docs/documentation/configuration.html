Configurations

The config object

{
  buildUiSelects: true, // [true, false]
  buildUiCheckboxesRadios: true, // [true, false]
  buildUiInputsFile: true, // [true, false]
  selects: {
    element: 'select', // [string (css selector)]
    selectByArrows: true, // [true, false]
    selectByDigit: true, // [true, false]
    autoHeight: true, // [true, false]
    autoPositioning: true, // [true, false]
    selectors: {...},
    callbacks: {...}
  },
  checkboxesRadios: {
    element: 'input[type="checkbox"],input[type="radio"]', //[string (css selector)]
    labelPosition: 'right', // ['right', 'left']
    selectors: {...},
    callbacks: {...}
  },
  inputsFile: {
    element: 'input[type="file"]', // [string (css selector)]
    defaultInputText: 'No file selected', // [string]
    defaultButtonText: 'Select a file', // [string]
    multipleSelecteText: '[x] selected files', // [x] will be change by real amount,
    maxLengthFileName: 15, // [int]
    selectors: {...},
    callbacks: {...}
  }
};
Key Type Default Description
buildUiSelects boolean true

If is true, enable the plugin to build custom selects field.
buildUiCheckboxesRadios boolean true

If is true, enable the plugin to build custom checkbox and radio fields.
buildUiInputsFile boolean true

If is true, enable the plugin to build custom input file fields.
selects Object {...}

The place where you customize the custom select field options.
selects.elements string select

The css query of the select field which you want customize.
selects.selectByArrows boolean true

If true, enables the navigation for all select options by keyboard arrows.
selects.selectByDigit boolean true

If true, enables the navigation for all select options by keyboard digits.
selects.autoHeight boolean true

If true, the plugin will auto resize the options view.
selects.autoPositioning boolean true

If true, the plugin will positioning the option view dynamically according with vertical position of the select field.
selects.selectors Object {...}

The css selectors generated by the plugin for its dynamic elements. See more in selectors section.
selects.callbacks Object {...}

An object with custom callbacks methods. See more in callback section.
checkboxesRadios Object {...}

The place where you customize the custom checkbox and radio fields options.
checkboxesRadios.element string input[type="checkbox"], input[type="radio"]

The css query of the checkbox and/or radio field which you want customize.
checkboxesRadios.labelPosition string
[right, left]		 right

The position where the element label will appear.
checkboxesRadios.selectors Object {...}

The css selectors generated by the plugin for its dynamic elements. See more in selectors section.
checkboxesRadios.callbacks Object {...}

An object with custom callbacks methods. See more in callback section.
inputsFile Object {...}

The place where you customize the custom input file options.
inputsFile.element string input[type="file"]

The css query of the input file which you want customize.
inputsFile.defaultInputText string No file selected

The placeholder shown in empty input file.
inputsFile.defaultButtonText string Select a file

The text shown in input file button.
inptsFile.multipleSelecteText string [x] selected files

By default the name of file will be show into the input. But if more one files was selected the text defined here will be shown. The [x] will be change by real amount of files.
inputsFile.maxLengthFileName int 15

The number of characters that will be shown from selected file name.
inputsFile.selectors Object {...}

The css selectors generated by the plugin for its dynamic elements. See more in selectors section.
inputsFile.callbacks Object {...}

An object with custom callbacks methods. See more in callback section.
