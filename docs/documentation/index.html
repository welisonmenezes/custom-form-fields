Basic Usage

Instalation

Call the css file at the header:

<link rel='stylesheet' src='dist/css/custom-form-fields.min.css'></>

Call the javascript file before the close tag body:

<script src='dist/custom-form-fields.min.js'></script>

Initialization

<script>
  var cff = CustomFormFields.init({});
</script>

Where is applied

By default the plugins applies its functionalities over defaults elements selects, checkboxes, radios and input files

To customize it see selectors section.

The config object

You have to pass a configuration object as parameter when you call the CustomFormFields.init method.

If you pass a empty object the CustomFormFields will use the default configurations.

However, you can customize your configurations. Example:

<script>
var config = {
	selects: {
		selectByDigit: true,
		selectors: {
			selected: 'cff-selected'
		}
	},
	checkboxesRadios: {
		selectors: {
			checked: 'cff-is-checked'
		}
	}
};
var cff = CustomFormFields.init(config);
</script>
