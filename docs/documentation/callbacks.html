The select element callbacks

Key Type Default Description
beforeBuildSelects Function null

Called before the plugin build all selects. Its parameter receives the select elements.
afterBuildSelects Function null

Called after the plugin build all selects. Its parameter receives the ui select elements.
beforeConstroySelect Function null

Called before each ui select be built. Its paramerter receives the select element.
afterConstroySelect Function null

Called after each ui select be built. Its parameter receives the ui select element.
beforeDestroySelect Function null

Called before the ui select be destroyed. Its parameter receives the ui select element. The ui element is destroyed by destroy() method. See more at Method section.
afterDestroySelect Function null

Called after the ui select be destroyed. Its parameter receives the select element. The ui element is destroyed by destroy() method. See more at Method section.
beforeAddNewOption Function null

Called before the new option be added. Its parameter receives the new option element. The new option is added by addNewOption method. See more at Methods section.
afterAddNewOption Function null

Called after the new option be add. Its parameter receives the new ui option element. The new option is added by addNewOption method. See more at Methods section.
beforeOpenSelect Function null

Called before the select be opened. Its parameter receives the ui select element.
afterOpenSelect Function null

Called after the select be opened. Its parameter receives the ui select element.
beforeCloseSelects Function null

Called before the select the be closed. Its parameter receives every ui select element is opened.
afterCloseSelects Function null

Called after the select be closed. Its parameter receives every ui select element was opened.
beforeSelectItem Function null

Called before a option be selected. Its parameter receives the ui select element.
afterSelectItem Function null

Called after a option be selected. Its parameter receives the ui select element.
beforeDeselectItem Function null

Called before a option be deselected. Its parameter receives the ui select element.
afterDeselectItem Function null

Called after a option be deselected. Its parameter receives the ui select element.

The checkbox and radio elements callbacks

Key Type Default Description
beforeBuildInputs Function null

Called before build all custom checkbox and radio inputs. Its parameter receives the checkbox and radio elements
afterBuildInputs Function null

Called after build all custom checkbox and radio inputs. Its parameter receives the ui checkbox and radio elements.
beforeConstroyInput Function null

Called before constroy each checkbox or radio input. Its parameter receives the checkbox or tthe radio element.
afterConstroyInput Function null

Called after constroy each checkbox or radio input. Its parameter receives the ui checkbox or the ui radio element.
beforeDestroyInput Function null

Called before destroy the ui checkbox or ui radio element. Its parameter receives the ui checkbox or the ui radio element. The ui element is destroyed by destroy() method. See more at Method section.
afterDestroyInput Function

Called after destroy the ui checkbox or ui radio element. Its parameter receives the checkbox or the radio element. The ui element is destroyed by destroy() method. See more at Method section.
beforeCheckInput Function null

Called before the checkbox or the radio element being checked. Its parameter receives the checkbox or the radio element.
afterCheckInput Function null

Called after the checkbox or the radio element being checked. Its parameter receives the checkbox or the radio element.
